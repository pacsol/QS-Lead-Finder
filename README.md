# QS Nexus

A lead generation and market intelligence platform for self-employed Quantity Surveyors in the UK, powered by Gemini Search and Maps grounding.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Google Gemini API (@google/genai)
- Recharts

## Getting Started

```bash
npm install
npm run dev
```

The app runs on `http://localhost:3000`.

## Environment Variables

Create a `.env.local` file:

```
GEMINI_API_KEY=your_api_key_here
```
